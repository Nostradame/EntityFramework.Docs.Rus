Приступая к работе на .NET Core (Windows, OSX, Linux и т.д.)
=============================================================

Эти 101 учебные пособия не требуют предварительных знаний о Entity Framework (EF) или Visual Studio. Они проведут вас шаг за шагом через процесс создания простого приложения, которое запрашивает данные из базы данных и сохраненяет их.

EF может использоваться на всех платформах (Windows, Linux, OSX и т.д.), которые поддерживают .NET Core.
