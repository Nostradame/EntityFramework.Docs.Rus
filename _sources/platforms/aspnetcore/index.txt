Приступая к работе на ASP.NET Core
====================================

Эти 101 учебные пособия не требуют предварительных знаний о Entity Framework (EF) или Visual Studio. Они проведут вас шаг за шагом через процесс создания простого приложения, которое запрашивает данные из базы данных и сохраненяет их.

Entity Framework может создать модель, основанную на существующей базе данных, или создать базу данных для вас на основе вашей модели. Следующие учебные пособия продемонстрируют оба этих подхода используя приложение ASP.NET Core.
