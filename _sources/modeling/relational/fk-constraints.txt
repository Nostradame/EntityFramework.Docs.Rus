Ограничения внешних ключей
==============================

Ограничения внешних ключей вводятся для любых отношений в модели.

Соглашения
---------------

По соглашению, ограничения внешних ключей именуются ``FK_<имя зависимого типа>_<имя главного типа>_<имя свойства внешнего ключа>``. Для составных внешних ключей ``<имя свойства внешнего ключа>`` составляется из списка имен свойств внешних ключей, разделенных нижним подчеркиванием.

Аннотации данных
------------------------

Имена ограничений внешних ключей не могут быть настроены при помощи аннотации данных.

Fluent API
-------------------

Вы можете использовать Fluent API для настройки имен ограничений внешних ключей для отношений.
