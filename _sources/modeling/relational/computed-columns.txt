Вычисляемые столбцы
====================

Вычисляемый столбец — это столбец, значение которого вычисляется в базе данных. Вычисляемый столбец может использовать другие столбцы в таблице, чтобы рассчитать свое значение.

Соглашения
----------

По соглашению вычисляемые столбцы не создаются в модели.

Аннотации данных
----------------

Вычисляемые столбцы не могут конфигурироваться аннотациями данных.

Fluent API
----------

Вы можете использовать Fluent API чтобы определить, какие свойства должны сопоставляться вычисляемым столбцам.
