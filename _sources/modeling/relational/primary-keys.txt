Первичные ключи
==================

Ограничение первичного ключа вводится для ключа каждого типа объекта.

Соглашения
----------

По соглашению, первичный ключ в базе данных будет называться ``PK_<имя типа>``.

Аннотации данных
----------------

Никакие аспекты первичных ключей, специфичные для реляционных баз данных не могут быть сконфигурированы при помощи аннотации данных.

Fluent API
----------

Вы можете использовать Fluent API для настройки имени ограничения первичных ключей в базе данных.
