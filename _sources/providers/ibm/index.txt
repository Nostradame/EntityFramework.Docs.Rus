Серверы данных IBM
=======================

This database provider allows Entity Framework Core to be used with IBM Data Servers. Проблемы, вопросы, и т.д. могут быть размещены на `.Net Development with DB2 and IDS forum <https://www.ibm.com/developerworks/community/forums/html/forum?id=11111111-0000-0000-0000-000000000467>`_

.. attention::
    This provider currently only supports the Entity Framework Core RC1 pre-release.

.. contents:: `В этой статье:`
    :depth: 2
    :local:

Install
-------

Install the `EntityFramework.IBMDataServer NuGet package <https://www.nuget.org/packages/EntityFramework.IBMDataServer>`_.

.. code-block:: text

    PM>  Install-Package EntityFramework.IBMDataServer -Pre
