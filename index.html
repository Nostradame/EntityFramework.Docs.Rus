Entity Framework Core¶

Entity Framework (EF) Core это легкая и расширяемая версия популярной технологии доступа к данным платформы Entity Framework.

EF Core это объектно-реляционный сопоставитель(O/RM) который позволяет разработчикам .NET работать с базой данных при помощи объектов .NET. Это значительно уменьшает необходимость написания кода доступа к данным, который обычно необходим разработчикам. EF Core поддерживает множество движков баз данных, подробнее смотрите Провайдеры баз данных.

Если вам нравиться начинать обучение с написания кода, мы рекомендуем одно из наших руководств Приступая к работе, чтобы вы начали изучать EF Core.

Установка Entity Framework Core¶

Установите пакет NuGet для того провайдера базы данный, которого вы хотите использовать. Подробнее смотрите раздел Провайдеры баз данных.

PM>  Install-Package Microsoft.EntityFrameworkCore.SqlServer

Модель¶

В EF Core доступ к данным осуществляется с помощью модели. Модель состоит из классов сущностей и производного контекста, который представляет сеанс связи с базой данных, что позволяет запрашивать и сохранять данные. Чтобы узнать больше смотрите раздел Создание модели.

Вы можете создать модель из существующей базы данных, вручную написать код, соответствующий вашей базе данных, или использовать EF Migrations для создания базы данных из вашей модели (и развивать ее по мере изменения вашей модели).

using Microsoft.EntityFrameworkCore;
using System.Collections.Generic;

namespace Intro
{
    public class BloggingContext : DbContext
    {
        public DbSet<Blog> Blogs { get; set; }
        public DbSet<Post> Posts { get; set; }

        protected override void OnConfiguring(DbContextOptionsBuilder optionsBuilder)
        {
            optionsBuilder.UseSqlServer(@"Server=(localdb)\mssqllocaldb;Database=MyDatabase;Trusted_Connection=True;");
        }
    }

    public class Blog
    {
        public int BlogId { get; set; }
        public string Url { get; set; }

        public List<Post> Posts { get; set; }
    }

    public class Post
    {
        public int PostId { get; set; }
        public string Title { get; set; }
        public string Content { get; set; }

        public int BlogId { get; set; }
        public Blog Blog { get; set; }
    }
}

Выполнение запросов¶

Экземпляры классов сущностей извлекаются из базы данных с помощью Language Integrated Query (LINQ). Чтобы узнать больше смотрите Запрос данных.

using (var db = new BloggingContext())
{
    var blogs = db.Blogs
        .Where(b => b.Rating > 3)
        .OrderBy(b => b.Url)
        .ToList();
}

Сохранение данных¶

Данные создаются, удаляются и изменяются в базе данных при помощи экземпляров классов сущностей. Чтобы узнать больше смотрите Сохранение данных.

2

using (var db = new BloggingContext())
{
    var blog = new Blog { Url = "http://sample.com" };
    db.Blogs.Add(blog);
    db.SaveChanges();
}
