Серверы данных IBM¶

Этот провайдер базы данных позволяет использование Entity Framework Core вместе с IBM Data Servers. Проблемы, вопросы, и т.д. могут быть размещены на .Net Development with DB2 and IDS forum

Внимание

Этот провайдер в настоящее время поддерживает только Entity Framework Core RC1 pre-release.

Установка¶

Установите NuGet пакет EntityFramework.IBMDataServer.

PM>  Install-Package EntityFramework.IBMDataServer -Pre
