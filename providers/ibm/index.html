Серверы данных IBM¶

This database provider allows Entity Framework Core to be used with IBM Data Servers. Проблемы, вопросы, и т.д. могут быть размещены на .Net Development with DB2 and IDS forum

Внимание

This provider currently only supports the Entity Framework Core RC1 pre-release.

Install¶

Install the EntityFramework.IBMDataServer NuGet package.

PM>  Install-Package EntityFramework.IBMDataServer -Pre
