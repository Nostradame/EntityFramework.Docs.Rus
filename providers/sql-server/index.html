Microsoft SQL Server¶

Этот провайдер базы данных позволяет использование Entity Framework Core вместе с Microsoft SQL Server (включая SQL Azure). Провайдер поддерживается в рамках GitHub проекта EntityFramework.

В этой статье:

  • Установка
  • Начало работы
  • Поддерживаемые движки баз данных
  • Поддерживаемые платформы

Установка¶

Установите NuGet пакет Microsoft.EntityFrameworkCore.SqlServer.

PM>  Install-Package Microsoft.EntityFrameworkCore.SqlServer

Начало работы¶

Следующие ресурсы помогут вам начать работу с этим провайдером.
  • Приступая к работе на Full .NET (Console, WinForms, WPF, etc.)
  • Приступая к работе на ASP.NET Core
  • Пример приложения UnicornStore

Поддерживаемые движки баз данных¶

  • Microsoft SQL Server (2008 и далее)

Поддерживаемые платформы¶

  • Full .NET (4.5.1 и далее)
  • .NET Core
  • Mono (4.2.0 и далее)

Внимание

С помощью этого поставщика на Mono будет использовать реализации клиента Mono SQL, который имеет ряд известных проблем. Например, он не поддерживает защищенные соединения (SSL).
