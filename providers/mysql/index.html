MySQL (скоро)¶

Мы работаем с командой MySQL чтобы сделать доступным провайдер MySQL. Пожалуйста, направляйте любые вопросы об этом поставщике, включая сроки выпуска, в Проект MySQL.

DevArt dotConnect предлагают платного провайдера MySQL, подробнее смотрите Devart (MySQL, Oracle, PostgreSQL, SQLite, DB2, SQL Server, and more).
