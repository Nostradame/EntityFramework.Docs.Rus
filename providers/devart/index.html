Devart (MySQL, Oracle, PostgreSQL, SQLite, DB2, SQL Server, и другие)¶

Devart это провайдер сторонних разработчиков, которые предлагают провайдеров для многих баз данных. Узнайте больше на devart.com/dotconnect.

Внимание

В настоящее время Devart dotConnect поддерживает только Entity Framework Core RC2 pre-release.

В этой статье:

  • Только платные версии

Только платные версии¶

Devart dotConnect это сторонний коммерческий провайдер. Поддержка Entity Framework доступна только в платной версии dotConnect.

Установка¶

Инструкцию по установке смотрите в документации Devart dotConnect.

Начало работы¶

Для начала работы смотрите документацию Devart dotConnect Entity Framework.
